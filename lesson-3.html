Classes

So far we have learned about the "id" attribute of HTML tags. It sets an id for JavaScript to use to access elements. However, what if you want to deal with groups of tags? For example what if you want to set the "onclick" of all of your buttons and text boxes? Enter the class attribute of an HTML tag. It is set in the same way "id" is set, but uses the "class" attribute and denotes that element as part of a group. On "bind" - Optional

As we learned last week, alert is a function that takes one argument, a string to print in the alert box. The "bind" function (as seen above) is used to create a new function with the arguments pre-filled. Last week we had to call a fucntion like the below in order to pass alert its argument.

function fn1() { alert("Some string"); }

However, that is a lot of extra code to be able to use alert with no-arguments. We created a function to pre-fill the string. "bind" does the same thing, creating an anonymous function and asigning the arguments. "bind" takes N+1 arguments. The first argument we need to set to "undefined". The last N arguments fill the N arguments of the function you are binding. Here "alert" has 1 argument, so bind take 1+1 arguments. The first is "undefined" and the second is the string we wish to pass to alert.

Now lets add some CSS. We can change the background-color of our buton and input from above. Lets make it pink, because pink is objectively the best color. CSS starts with an element specifier (like a tag name, id, or class name) and then follows with a set of properties to apply to the element(s) that match the specifier. If multiple sets of properties match, CSS uses the most specific matcher. Copy the below code into your index.css file and rerun your app.

/* Specify all h1 headers using a "h1" tag specifier*/ h1 { background-color: red; } /* We can set properties for all elements in a class using the class name prefixed by a "."*/ .agroup { background-color: pink; } Try some other properties, like "border", "margin", and "padding". Bootstrap

In industry, we use libraries and other third-party code in order to hasten the development of out apps. It is important to know the basic elements so we can customize key features, but it is sometimes nice to take advatage of the work of other developers in the field. Bootstrap is a design library that gives a modern look to elements in your app, if you apply it. It is opensource, meaning you can see how it is made.

In order to use Bootstrap, you'll need to add some lines that link in that code. These lines are added to your "index.html" file. Add the fist line somewhere in the head section between the pair of "head" tags.

<head> ... ... <link rel="stylesheet" type="text/css" href="bootstrap-dist/css/bootstrap.css"> ... ... </head>

Add the next two lines just before the other script tags in the body of your index.html document.

<body> ... ... <script src="js/jquery-3.1.1.min.js"></script> <script src="bootstrap-dist/js/bootstrap.min.js"></script> <script ... ... </body>

Congradulations! Now you can use Bootstrap. Bootstrap automatically styls things when they are part of predefined classes. All you have to do is add classes to your elements that bootstrap knows about. One class that bootstrap styles is "btn-danger" which creates a bright red button. Lets try it:

<button class="btn-danger">Danger!</button> Spend some time and see what else Bootstrap can do. More information on Bootstrap:getbootstrap.com/css/
